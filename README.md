# Museo Vivo - Backend

Este proyecto es el backend del sistema *Museo Vivo*, desarrollado en **Java Spring Boot**.
Permite gestionar obras de arte (`Artwork`) y cuadros (`Cuadro`) a través de una API RESTful.

## Tecnologías
- Java 17
- Spring Boot
- Maven
- Spring Data JPA
- H2 / Postgres  

## Ejecución
```bash
mvn spring-boot:run
